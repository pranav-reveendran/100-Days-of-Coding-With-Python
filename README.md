## Overview

This repository documents my journey through the "100 Days of Code: The Complete Python Pro Bootcamp" course. Each day's project is organized in its own dedicated branch for easy navigation and portfolio showcase.

**Repository Structure:** Each day (Day 1 through Day 100) has its own branch named `day-01`, `day-02`, ..., `day-100`. Click on any day below to view the code!

## How to Navigate This Repository

1. **Browse by Day:** Click on any day link above to view that day's branch
2. **Clone the Repository:** `git clone https://github.com/pranu98/100-Days-of-Coding-With-Python.git`
3. **Switch to a Specific Day:** `git checkout day-XX` (replace XX with day number, e.g., day-01)
4. **View All Branches:** `git branch -a`
