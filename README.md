## Overview

This repository documents my journey through the "100 Days of Code: The Complete Python Pro Bootcamp" course. Each day's project is organized in its own dedicated folder for easy navigation and portfolio showcase.

**Repository Structure:** Each day (Day 1 through Day 100) has its own folder named `day01`, `day02`, ..., `day100`. Click on any day below to view the code!

## How to Navigate This Repository

1. **Browse by Day:** Click on any day link above to view that day's folder
2. **Clone the Repository:** `git clone https://github.com/pranu98/100-Days-of-Coding-With-Python.git`
3. **Navigate to a Specific Day:** `cd dayXX` (replace XX with day number, e.g., day01, day15, day100)
4. **View All Days:** `ls -d day*`
